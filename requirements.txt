# 2. INSTALL PYTORCH (PILIH SALAH SATU)
#
#    a) CPU-ONLY (Recommended untuk development)
#       pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
#
#    b) CUDA 12.1 (RTX 4000-4090, H100, etc)
#       pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
#
#    c) CUDA 11.8 (RTX 3000-3090, A100, etc)
#       pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#
#    d) ROCm 5.7 (AMD GPU)
#       pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/rocm5.7
#
#    Atau gunakan PyTorch selector: https://pytorch.org/get-started/locally/
#
# 3. INSTALL DEPENDENCIES DARI FILE INI
#    pip install -r requirements.txt
#
